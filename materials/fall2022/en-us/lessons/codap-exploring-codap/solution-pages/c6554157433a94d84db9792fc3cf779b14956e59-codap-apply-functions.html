Applying Functions

In order to answer each of the questions below, you must first select test in Animals Starter File, and then choose Edit formula from the drop-down menu that appears. You will be typing into the Edit Formula box.

  • Enter sqrt(16) into the Edit Formula box.

3 How many arguments does sqrt expect? 1

4 What type of argument does the function expect? (Number? String? Boolean?) Number.

5 What type of output does sqrt produce? Number.

6 What did the function evaluate to? 4

  • Now, enter sqrt(Weight).

7 What do you observe? CODAP finds the square root of each animal’s weight

8 What happens when you enter sqrt(Legs)? CODAP finds the square root of each animal’s legs for sqrt(Legs).

9 What about sqrt(Name)? CODAP produces an error message because it cannot find the square root of a string, only a number.

  • Enter stringLength(Name) into the test column.

10 What do you observe? 7 appears, CODAP finds the amount of letters in each animal’s name

11 How many arguments does stringLength expect? 1

12 What type of argument does stringLength expect? String

13 Does stringLength produce a Number? String? Boolean? Number

Sort Transformer

Make sure you’re logged into CODAP with the Animals Dataset open. Open the Transformers plugin, and select Sort.

Creating a Sort Transformer

  • Select Sort from the drop-down menu of Transformers.

  • We want to give each Transformer a useful name, one which clearly describes what that Transformer does. Generally, a brief hyphenated description works - for instance: order-by-pounds.

1 What would be a good name for a Transformer that organizes the table alphabetically by animals' names?

order-by-name.

  • Type in the name you wrote above.

2 Which dataset do you want CODAP to apply this Transformer to? Animals Dataset

  • Select the appropriate dataset from the drop-down menu.

  • Select Name as the attribute to sort.

3 Which direction will you select? Why?

"Ascending" sorts from A to Z, so I will select that.

Applying and Saving the Sort Transformer

4 Apply the Transformer. What happens?

A copy of the original dataset is created.

5 What do you notice about the name of the new dataset?

It has the Transformer’s customized name at the front, with the name of the original dataset in parentheses.

6 What happens when you save a Transformer?

When I save a Transformer, a small box appears that has my Transformer’s name at the top. I will now be able to use the Transformer again, without inputting all of the different specifications.

7 How is saving a Transformer different from saving a dataset?

Saving a dataset saves the table, but not the operations performed on it. Saved tables appear on the drop-down menu of datasets to transform. Saving a transformer lets us re-use an operation.

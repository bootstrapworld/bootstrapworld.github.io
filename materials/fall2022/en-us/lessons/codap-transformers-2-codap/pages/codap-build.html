Build Attribute

Make sure you’re logged into CODAP with the Animals Dataset open.

Creating a Build Attribute Transformer

1 Now select Build Attribute in the Transformer plugin.

2 Enter build-is-heavy as the name. What does this name tell you about the Transformer we are creating?

3 Select animals-dataset.

4 What would be an appropriate name for our new attribute?

5 Examine the contract. A domain is provided (row), but not a range. What is the desired output for build-is-heavy?

6 Let’s write a purpose statement. What do you think we want a Transformer called build-is-heavy to do?

7 Enter Weight > 40 as the expression.

8 Apply the Transformer. To define the Transformer for future use, select Save.

More Building

9 Now, try creating a Transformer called updated-age. This Transformer will give the animals' ages one year from today. How many animals are 9 years-old one year from today?

Dots for Value Ranges - Explore

Open the Dots for Value Ranges Starter File and look at the code.

What do you notice? What do you wonder?

1 How is this program similar to the last one we looked at?

It uses a helper function and the image-scatter-plot function.

2 How is the code for weight-dot different from other functions we’ve seen?

The piecewise function produces dot sizes based on which interval the value falls within!

3 Can you explain how this code works?

It takes in a row and checks to see which range the weight falls within.

If the weight falls within a given range, it makes a dot of the corresponding size.

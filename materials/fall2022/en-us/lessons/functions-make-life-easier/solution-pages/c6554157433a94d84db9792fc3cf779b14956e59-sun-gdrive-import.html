Let’s Define Some More New Functions!

1 Let’s define a function sun to write SUNSHINE in whatever color and size we give it!

If I say sun​(​5, "blue"​), what would our actor need to say?

text​(​"SUNSHINE", 5, "blue"​)

Let’s write a few more examples:

sun​(​20, "red"​) → text​(​"SUNSHINE", 20, "red"​)

sun​(​99, "green"​) → text​(​"SUNSHINE", 99, "green"​)

sun​(​50, "pink"​) → text​(​"SUNSHINE", 50, "pink"​)

What changes in these examples? Name your variable(s): size, color

Let’s define our function using the variable.

fun sun​(​size, color​): text​(​"SUNSHINE", size, color​) end

2 Let’s define a function me to generate your name in whatever size and color we give it!

If I say me​(​18, "gold"​), what would our actor need to say?

text​(​"Malik", 18, "gold"​)

Let’s write a few more examples:

me​(​22, "green"​) → text​(​"Malik", 22, "green"​)

me​(​10, "red"​) → text​(​"Malik", 10, "red"​)

me​(​40, "pink"​) → text​(​"Malik", 40, "pink"​)

What changes in these examples? Name your variable(s): size, color

Let’s define our function using the variable.

fun me​(​size, color​): text​(​"Malik", size, color​) end

3 Let’s define a function gr to build a solid, green rectangle of whatever length and width we give it!

If I say gr​(​10, 80​), what would our actor need to say?

rectangle​(​10, 80, "solid", "green"​)

Let’s write a few more examples:

gr​(​50, 50​) → rectangle​(​50, 50, "solid", "green"​)

gr​(​10, 50​) → rectangle​(​10, 50, "solid", "green"​)

gr​(​98, 15​) → rectangle​(​98, 15, "solid", "green"​)

What changes in these examples? Name your variable(s): length, width

Let’s define our function using the variable.

fun gr​(​length, width​): rectangle​(​length, width, "solid", "green"​) end
